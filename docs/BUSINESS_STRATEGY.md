## 0. エグゼクティブサマリー

PRismは、PR会社・広告代理店向けの**競合PR戦略分析ダッシュボード**である。既存ツール（PR Analyzer / Meltwater等）が「過去の露出量を報告するクリッピングツール」に留まる中、PRismは**「競合の勝ちパターンを因数分解し、次の打ち手を示唆する戦略立案ツール」**という未開拓ポジションを狙う。

PESOモデル（Paid / Earned / Shared / Owned）を分析フレームワークの軸とし、月額¥29,800〜の「稟議不要」価格帯で、PLG（Product-Led Growth）による低CACでの顧客獲得を目指す。

**核心仮説**: 中堅・中小PR会社は「次に何をすべきか」の示唆に飢えているが、既存ツールは高価で報告用途に偏っている。この**インサイト・ギャップ**がPRismの市場機会である。

### KGI（12ヶ月目標）

| 指標 | 目標値 |
|---|---|
| MRR（月間定期収益） | ¥1,000,000 |
| 有料顧客数 | 25社 |

| 月次チャーン | 3%以下 |

### 1.3 市場の構造的変化（PRismに追い風となるトレンド）

1. **AVE（広告換算価値）への批判の高まり**: AMEC（国際メディア評価協会）がAVEを公式に否定。「効果測定の新基準」への移行期
2. **PR会社のDX遅延**: 営業・企画は属人的スキルに依存。データドリブンなツールの浸透率はマーケティング部門の1/3以下
3. **中小PR会社の増加**: フリーランスPRコンサルタント、ブティック型事務所の増加。しかし既存ツールは高額でリーチできていない
4. **日本SaaS市場の成長**: 2025年USD 122億、CAGR 13.5%で拡大中。SaaSへの抵抗感は年々低下

## 2. 差別化戦略

### 2.1 PRismのコアバリュー

既存ツールが「**What happened?**（何が起きたか）」を報告するのに対し、PRismは「**Why it worked? → What's next?**（なぜ効いたか → 次に何をすべきか）」を示す。

```
┌───────────────────────────────────────────────────────────┐
│  既存ツール: What happened?                                │
│  「先月の露出量は1,200件、広告換算価値は3,000万円でした」    │
│                                                           │
│     ↓  PRismが埋めるギャップ                                │
│                                                           │
│  PRism: Why it worked? → What's next?                     │
│  「競合Aは"成分訴求×専門家コラボ"のPESOパターンで            │
│   Earned露出が3倍に。御社も同パターンの横展開が有効です」     │
└───────────────────────────────────────────────────────────┘
```

### 2.2 4層のMoat（競争優位の壁）設計

PRismの防御壁は段階的に構築する。

| 層 | Moat | 内容 | 構築時期 |
|---|---|---|---|
| **第1層** | 価格の壁 | 稟議不要の¥29,800〜。大手ツールが価格を下げにくい構造を利用 | Day 1 |
| **第2層** | UXの壁 | Next.js + shadcn/uiによる軽快なSPA体験。「毎日開きたくなるUI」 | Phase 1 |
| **第3層** | データの壁 | 蓄積された競合PR分析データ。業界別ベンチマーク。スイッチングコスト上昇 | Phase 2〜 |
| **第4層** | AIの壁 | 蓄積データで訓練したPR戦略推論モデル。「PRismでしか出せない示唆」 | Phase 3〜 |

### 2.3 差別化ポイントの整理

| 差別化軸 | 詳細 | 顧客にとっての意味 |
|---|---|---|
| **戦略立案特化** | 「結果報告」ではなく「勝ちパターンの因数分解」 | クライアントへの提案品質が上がる |
| **PESOフレームワーク** | 4象限での構造的分析。業界初のPESO特化ダッシュボード | PR戦略の「抜け漏れ」が一目で分かる |
| **モダンUI** | Next.js + shadcn/uiのSPA。3分で朝のトレンド把握 | 日常業務に溶け込む。ツール定着率が上がる |
| **AI示唆** | グラフを見せるだけでなく「次のアクション」を提案 | 分析スキルに依存せず、ジュニアでも戦略提案可能 |
| **稟議不要価格** | 月額¥29,800〜。部門経費で即決 | 導入にかかるリードタイムが数日 |

## 3. ビジネスモデル・価格戦略

### 3.1 Value Metric（課金の軸）

**「分析対象ブランド登録数」** を課金軸とする。

**選定理由**:
- PR会社はクライアント増加 = ブランド枠の必要数増加。**事業成長とアップセルが自然に連動**
- 「ユーザー数課金」は社内浸透のブレーキになるため回避
- 「API呼び出し数」は予測困難で顧客の不安要因になるため回避

| **年額（税抜）** | ¥298,000（2ヶ月分お得） | ¥498,000（2ヶ月分お得） | 個別見積 |

**価格設計のロジック**:

- **Starter ¥29,800**: 「自社＋主要競合4社」の定点観測。お試しに最適。部門予算で即決可能
- **Pro ¥49,800**: 稟議不要ライン（5万円未満）の上限ギリギリを突く。即決かつ最大限のARPU
- **Agency ¥150,000〜**: 組織全体導入。コンサルティング的価値まで含め、Meltwaterの1/3以下で訴求

**年額ディスカウント**: 年契約は2ヶ月分無料（実質16.7%割引）。キャッシュフロー安定化とチャーン低減を狙う。

**無料トライアル**: 14日間、クレカ登録不要。トライアル中にブランド3社まで分析可能。「価値を体感してから課金」のPLGフロー。


### 4.2 North Star Metric

**「週次アクティブブランド分析数」（WAB: Weekly Active Brand Analyses）**

PRismの核心価値は「競合ブランドのPR戦略を分析する行為」そのもの。この指標が伸びている限り、課金・リテンション・口コミはすべて後からついてくる。

| 時期 | WAB目標 | 意味 |
|---|---|---|
| 6ヶ月目 | 50 | 週に50ブランドが分析されている |
| 12ヶ月目 | 200 | PMFの証拠 |
| 24ヶ月目 | 1,000 | スケールフェーズ |

### 4.3 アクティベーション設計

トライアルユーザーが「価値を実感する瞬間」を5分以内に到達させる。

**アクティベーション・チェックリスト（ユーザーの最初の5分）**:

| ステップ | 所要時間 | ユーザーの体験 |
|---|---|---|
| 1. サインアップ | 30秒 | メール＋パスワードのみ。Google OAuth対応 |
| 2. 最初のブランド登録 | 60秒 | ブランド名を入力→候補をサジェスト→1クリック登録 |
| 3. デモデータ表示 | 即座 | 登録直後にサンプルデータでPESOダッシュボードが表示される |
| 4. 競合ブランド追加 | 60秒 | 「このブランドの競合を追加しましょう」のガイド |
| 5. **Aha Moment** | — | 「競合と自社のPESOバランスの違い」が視覚的に明らかになる |

## 5. Go-to-Marketロードマップ

### Phase 1: 「0→1」デザインパートナー獲得（Month 1〜3）

**目標**: 熱狂的な初期ユーザー3〜5社を獲得し、PMFの手応えを掴む

**戦略: コンシェルジュ・オンボーディング**

プロダクトが完璧になる前に売りに行く。エンジニアリング力を「コンシェルジュ」として使い、顧客が欲しいデータを手動で用意して価値を証明する。

| アクション | 詳細 |
|---|---|
| **ターゲット選定** | 都内のブティック型PR会社（5〜20名規模）、D2Cスタートアップ（コスメ・アパレル）のPR担当 |
| **アウトバウンド** | X / YOUTRUSTで「PR・広報」担当者を検索→DMでデモ打診。月20件送付目標 |
| **壁打ち** | スタートアップ系ミートアップ（Startup Hub Tokyo等）に参加。プロトタイプの画面を見せて反応検証 |
| **コンシェルジュ提案** | 「御社の競合3社の直近半年分のPRデータを入れておきました。月額1万円で使い放題にするので、週1回フィードバックMTGをさせてください」 |
| **KPI** | デザインパートナー3社獲得、週次MTG実施率100% |

**Phase 1 の Exit Criteria**: 3社以上が「このツールなしでは仕事に戻れない」と発言

### Phase 2: 「1→10」PMF達成（Month 4〜8）

**目標**: 再現可能な獲得チャネルを見つけ、MRR ¥200,000到達

**戦略: Data-Led Content Marketing**

SaaS内のデータから質の高い分析記事を量産し、インバウンドリード獲得の仕組みを構築する。

| アクション | 詳細 |
|---|---|
| **コンテンツ発信** | noteで業界別PR分析レポート（月2本）。例：「コスメ業界のPRトレンド2026」 |
| **CTA設計** | 記事末尾に「この分析はPRismで5分で作成。無料トライアルはこちら」＋Calendlyリンク |
| **SEO** | 「PR 競合分析 ツール」「PESO分析 方法」等のロングテールKWで記事作成 |
| **課金開始** | Stripe連携でセルフサーブ課金。Starter / Proプランを正式展開 |
| **NPS測定** | 月次でNPS調査。Detractor（批判者）には即座にCS対応 |
| **KPI** | MRR ¥200K、有料顧客6社、NPS 40+ |

**Phase 2 の Exit Criteria**: 月間新規トライアル20件以上、PQL→有料転換率20%以上

### Phase 3: 「10→100」スケールアップ（Month 9〜18）

**目標**: PLGエンジンの本格稼働、MRR ¥1,000,000到達

**戦略: PLG × パートナーシップ**

| アクション | 詳細 |
|---|---|
| **セルフサーブ強化** | オンボーディングウィザードの最適化。5分以内のTime-to-Value達成 |

| **競合の模倣** | 低 | 低 | 🟢 低 | 4層Moat設計。データの壁＋AI の壁で時間差を作る |

## 付録A: 用語集

| 用語 | 定義 |
|---|---|
| **PESO** | Paid（広告）、Earned（パブリシティ）、Shared（SNS）、Owned（自社メディア）の4分類 |
| **AVE** | Advertising Value Equivalency（広告換算価値）。メディア露出を広告費に換算する旧来指標 |
| **SoV** | Share of Voice。業界内での話題量シェア |
| **PLG** | Product-Led Growth。プロダクト体験を通じて顧客獲得・拡大する成長戦略 |
| **PQL** | Product-Qualified Lead。プロダクト内の行動データから「購買意欲が高い」と判定されたリード |
| **NRR** | Net Revenue Retention。既存顧客ベースからの収益維持率（100%超ならアップセルが解約を上回る） |
| **WAB** | Weekly Active Brand Analyses。PRism固有のNorth Star Metric |
| **MRR** | Monthly Recurring Revenue。月間定期収益 |
