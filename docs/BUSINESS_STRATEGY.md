# PRism ビジネス戦略

> 経営・事業開発向け：競合分析、差別化戦略、価格プラン、Go-to-Marketロードマップ

---

## 1. 市場分析・競合

### 1.1 日本市場における主な競合

#### PR Analyzer（PRアナライザー） / ビルコム株式会社

- **特徴**: 日本のPR効果測定ツールの「超・王道」。テレビ、新聞、雑誌、Webメディア、SNSの露出を網羅的にクリッピング（収集）し、自社と競合の露出量や「広告換算価値（AVE）」を比較
- **弱点**: エンタープライズ向けで高額（月額数万円〜数十万円）。「広告換算価値」という古い指標に重きを置いており、現場の「次に何のSNS施策を打つべきか」という具体的なインサイトには直結しづらい

#### Meltwater（メルトウォーター） / Cision（シジョン）

- **特徴**: 外資系グローバルプラットフォーム。世界中のニュースとSNSを監視し、競合との「Share of Voice（話題量シェア）」をダッシュボードで可視化
- **弱点**: 多機能すぎて画面が複雑、導入コストも高い。「特定のブランドの勝ちパターンだけをサクッと分析したい」というニーズにはオーバースペック

#### Social Insight（ソーシャルインサイト） / 株式会社ユーザーローカル

- **特徴**: SNS（X、Instagram、TikTokなど）のアカウント分析や口コミ分析に特化。競合アカウントとの比較機能が強力
- **弱点**: SNS分析ツールのため、プレスリリースやメディア露出（パブリシティ）といった「PR全体」の文脈を横断して一元管理することには向いていない

### 1.2 市場の機会

既存サービスは「大企業が、自社がどれだけメディアに出たか（クリッピング・広告換算）を偉い人に報告するためのツール」になっている傾向がある。

**空いているポジション**:
- 中堅・中小PR会社向けの手頃な価格帯
- 「報告」ではなく「戦略立案」に特化
- モダンで軽量なUI/UX

## 2. 差別化戦略

| 差別化ポイント | 詳細 |
|---|---|
| **「報告用」ではなく「戦略立案用」** | 「結果の報告」ではなく「競合の勝ちパターンの因数分解」に特化 |
| **モダンで軽いUI** | Next.js + shadcn/uiのSPA。毎日使う担当者にとって大きな価値 |
| **AIによる示唆出し** | グラフを出すだけでなく「次のアクション」をAIが提案 |
| **稟議不要の価格帯** | 月額5万円未満で部門の経費で即決導入可能 |

### セールストーク（武器）

**「広告換算価値」からの脱却：**
> 「古いツールで『今月は1,000万円分の露出がありました』と報告して、クライアントは満足していますか？ 重要なのは『なぜそのブランドがウケたのか（勝ちパターンの因数分解）』と『次に何をすべきか』です。当アプリはそこ（インサイト）を可視化します」

**サクサク動くモダンな分析体験：**
> 「重くて複雑なエンタープライズツールではなく、Next.jsベースの圧倒的に軽快なUIを提供します。毎朝のコーヒーを飲みながら、3分で業界トレンドが把握できます」

## 3. ビジネスモデル・価格戦略

### 3.1 ターゲットと予算枠

| ターゲット | 詳細 |
|---|---|
| 本命 | 中堅・中小のPR会社 / 広告代理店のプランニングチーム |
| 決裁ライン | 「現場の部門長が、クレジットカードで即決できる金額」（月額5万円未満） |

### 3.2 価格プラン

課金の軸（Value Metric）は**「分析対象のブランド登録数」**。PR会社はクライアントが増えるほど分析対象のブランド枠が必要になるため、事業成長に合わせて自然にアップセルが狙える。

| プラン | 月額 | ターゲット | 制限 | 提供価値 |
|---|---|---|---|---|
| **Starter** | ¥29,800 | 事業会社のインハウスPR担当 | ブランド：5社 / ユーザー：3名 | 「自社＋主要競合4社」の定点観測。お試し導入しやすい価格帯 |
| **Pro** | ¥49,800 | PR会社の1チーム・中小代理店 | ブランド：20社 / ユーザー：無制限 | 稟議不要ライン（5万円未満）のギリギリを狙い、即決を促す |
| **Agency** | ¥150,000〜 | 大手PR会社・総合代理店 | ブランド：無制限 / AI示唆レポート自動生成 | 組織全体での導入。クライアント向けレポートのPDFエクスポート機能 |

※ 契約ハードルを下げるため「14日間の無料トライアル（クレカ登録なし）」を必ず用意

### 3.3 収益性

技術選定（Next.js/Supabase）によるランニングコストの低さが武器。サーバー代が月数千円で済むなら、月額3万円のStarterプランが1社でも売れれば黒字化。

## 4. Go-to-Marketロードマップ

### Phase 1: 「0→1」デザインパートナー獲得（1〜5社）

システムを完成させる前に売りに行く。エンジニアリング力を「コンシェルジュ的」に使い、熱狂的な初期ユーザーを作る。

- **ターゲット**: 都内の小規模なブティック型PR会社、D2Cスタートアップ（コスメ・アパレル等）
- **戦略**: コンシェルジュ・オンボーディング
  - 自身がスクリプトを書いて顧客が欲しがっている競合のPRデータを収集し、SaaSの画面に入れて見せる
  - 「御社の競合3社の直近半年分のデータを入れておきました。これ、月額1万円で使い放題にするので、週1回フィードバックのミーティングをさせてくれませんか？」と提案
- **アクション**:
  - X（旧Twitter）やYOUTRUSTで「PR・広報担当」を検索し、直接DMでデモを打診
  - 都内のスタートアップ系ミートアップに顔を出し、プロトタイプの画面を見せて壁打ち

### Phase 2: 「1→10」PMF達成

初期ユーザーのフィードバックで「これがないと仕事にならない」という状態（PMF）ができたら、課金モデルを本格化。

- **ターゲット**: 中小規模の広告代理店、中堅事業会社のマーケティング部門
- **戦略**: データ起点のコンテンツマーケティング（Data-Led Growth）
  - SaaS内のデータを使って質の高い分析レポート（例：「〇〇業界のPRトレンド」）をnoteやZennで発信
  - 記事の最後に「この分析は自社開発のPR分析ツールで5分で出しました。デモ希望はこちら」とカレンダーリンクを貼りリード獲得
- **価格**: Starterプラン（約3万円）/ Proプラン（約5万円）を適用。Stripe連携で即時導入可能に

### Phase 3: 「10→100」スケールアップと全国展開

東京で実績が出たら、それを「権威」として地方の有力企業や全国ネットの代理店へ横展開。

- **ターゲット**: 地方の有力PR会社、大手企業の広報部、PRコンサルタント
- **戦略**: PLG（プロダクト・レッド・グロース）と代理店開拓
  - **セルフサーブ化**: 営業を介さず「14日間無料トライアル」で価値を感じられるUI/UXに磨き上げ
  - **ウェビナー共催**: 熱量の高い顧客と一緒に「データで勝つPR戦略」セミナーを開催
  - **紹介プログラム**: 既存ユーザーが別のPR担当者を紹介したら数ヶ月無料になるリファラル施策

### Phase 4: 「100→」エンタープライズ層攻略

月額数万円の積み上げで安定した収益基盤ができたら、競合（PR Analyzer等）の牙城である「大手企業」のリプレイスを狙う。

- **ターゲット**: ナショナルクライアント（誰もが知る大企業）、総合広告代理店
- **戦略**: セキュリティ要件のクリアとカスタマーサクセス
  - **エンタープライズ要件**: SSO（SAMLログイン）、IP制限、監査ログの出力機能
  - **高単価プラン**: 月額15万円〜30万円の「Agency/Enterpriseプラン」。専任サポートを付け、「データを使ったPR戦略のコンサルティング」まで巻き取る
