# PRism ブランドガイド

> PR × Prism：競合のPR戦略を"分光"し、勝ちパターンを可視化する

---

## 1. ネーミングの構造と意味

```
PR  +  ism   +  Prism
──    ───      ─────
広報   主義・手法   光を分解する多面体
```

「PRism」は3つの意味を1語に圧縮している。

- **PR（Public Relations）** — ドメイン領域がひと目で伝わる
- **Prism（プリズム）** — 複雑に混在するPR情報を、メディア区分（PESO）ごとに美しく"分光"するツールの比喩
- **-ism（〜主義）** — データドリブンなPRという「新しい方法論」を提唱するニュアンス

---

## 2. 表記ルール

| 用途 | 表記 | 備考 |
|---|---|---|
| 正式名称 | **PRism** | 先頭2文字「PR」のみ大文字。これが最も重要なルール |
| ロゴ内 | PRism | フォントウェイトでPRを強調し、ismを軽くする |
| 日本語併記時 | PRism（プリズム） | 初出時のみカタカナを添える |
| URL・SNS | @prism_pr / prism-pr | 小文字統一 + 「_pr」で領域を明示 |
| NG表記 | ~~PRISM~~, ~~Prism~~, ~~prism~~ | 全大文字はNSAの監視プログラムを連想させるため厳禁 |

## 5. ビジュアルアイデンティティ

### 5.1 ロゴコンセプト

```
  ◇
 PRism
```

- プリズム（三角形 or 多面体）のシンボルマーク＋ワードマーク
- 「PR」部分を太字（Bold）、「ism」を細字（Light）にすることで視覚的にPRを浮き立たせる
- シンボルマークからは虹色の光が分散するモチーフ（分光のメタファー）

### 5.2 カラーパレット

| 役割 | カラー | Hex | 意図 |
|---|---|---|---|
| Primary | ディープインディゴ | `#4F46E5` | 信頼感・知的さ。SaaSの王道色 |
| Accent | プリズムバイオレット | `#8B5CF6` | プリズムの光を連想。モダンさを演出 |
| Secondary | エメラルド | `#10B981` | 成長・ポジティブなインサイトを表現 |
| Neutral | スレートグレー | `#64748B` | テキスト・UIの落ち着き |
| Background | ほぼ白 | `#F8FAFC` | クリーンで広々としたダッシュボード感 |

### 5.3 PESO区分カラー

アプリ全体で一貫して使用する。

| PESO区分 | カラー | Hex |
|---|---|---|
| Paid | オレンジ | `#F59E0B` |
| Earned | ブルー | `#3B82F6` |
| Shared | グリーン | `#10B981` |
| Owned | パープル | `#8B5CF6` |

### 5.4 タイポグラフィ

| 用途 | フォント | 理由 |
|---|---|---|
| ロゴ・見出し | **Inter** (Bold / Light) | モダンSaaSの定番。PRとismのウェイト差が映える |
| 本文（英数） | Inter (Regular) | ロゴと統一感を保つ |
| 本文（日本語） | Noto Sans JP (Regular) | Web表示の安定性と読みやすさ |

### 5.5 UIトーン

- ダッシュボードはダークモードをデフォルトに（データ分析ツールは暗い背景の方がグラフが映える）
- カードUI + 適度な余白で「情報過多」を感じさせない設計
- PESOの4区分には固定色を割り当て、アプリ全体で一貫させる

## 6. トーン＆ボイス

PRismは**「知的で頼れる参謀」**として振る舞う。

| 属性 | PRismらしい | PRismらしくない |
|---|---|---|
| 知的さ | データに基づく冷静な分析 | 曖昧な精神論 |
| 親しみ | カジュアルだが軽薄ではない | 堅苦しい敬語の羅列 |
| 自信 | 「分かります」と言い切る | 「〜かもしれません」と逃げる |
| 実用性 | 次のアクションを提示 | 分析結果だけ見せて終わり |

### 文体の例

**良い例（PRismらしい）：**
> CLAYGEのInstagram施策を分析しました。EARNED比率が競合平均より23%高く、UGC起点の拡散が成功要因です。インフルエンサーとの共創コンテンツを強化することを推奨します。

**悪い例（PRismらしくない）：**
> データを見ると、CLAYGEさんはインスタグラムが結構強いみたいですね！他社さんよりいいねが多い感じなので、頑張ってみてはいかがでしょうか？

## 8. ブランドの5原則

1. **名前がコンセプトを語る** — 「PRを分光する」という価値が名前に埋め込まれている
2. **PRの2文字で領域を即伝達** — 誰が見てもPR関連のツールだと分かる
3. **全大文字は絶対に使わない** — NSA連想の回避。常に「PRism」表記を徹底
4. **データに裏打ちされた自信** — 「かもしれない」ではなく「です」で語るトーン
5. **分析で終わらず、次の一手を示す** — ツールの存在意義は「示唆」にある
