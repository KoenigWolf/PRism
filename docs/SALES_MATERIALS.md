# PRism 営業資料

> 営業・BD向け：ピッチ、コピー、DM文面、デモ用データ

---

## 1. エレベーターピッチ（30秒）

> PR業界には「競合がどんな施策を打っているか」を調べる文化がありますが、多くの現場ではそれを人力でやっています。クリッピングツールはあっても「結果の報告」止まりで、「なぜあのブランドはウケたのか」「次に自分たちは何をすべきか」まで教えてくれるものはありません。
>
> PRismは、競合ブランドのPR施策をPESOモデルで構造化し、ブランド間の比較からAIが"次の一手"を提案する分析ダッシュボードです。毎朝3分で、業界の勝ちパターンが手に取るように分かります。

---

## 2. デモ・プレゼン時の冒頭トーク

> 「PRismは、プリズムが白色光を虹に分解するように、混沌としたPR市場の情報をPESO（Paid・Earned・Shared・Owned）に分解して、ブランドごとの勝ちパターンを可視化するダッシュボードです。今日は3分で、このツールの価値をお見せします。」

## 6. デモ用ダミーデータ（Prisma Seed用）

コスメ・美容業界（CLAYGEやNARSなど）を想定した、パッと見で「おっ、使えるな」と思わせるリアルなデータセット。

```typescript
// prisma/seed.ts の一部に組み込むダミーデータ例

const brandA = await prisma.brand.create({
  data: {
    tenantId: tenant.id,
    companyId: company.id,
    name: "CLAYGE（クレージュ）",
    category: "ヘアケア・スキンケア",
    prItems: {
      create: [
        {
          tenantId: tenant.id,
          sourceType: "Instagram",
          mediaType: "EARNED", // 第三者の口コミ
          channel: "美容系インフルエンサーA（フォロワー10万）",
          title: "春の新作シャンプー、ガチレビュー！",
          summary: "泥パック成分配合で頭皮スッキリ。香りがデパコス級と高評価。",
          engagementCount: 4500,
          publishedAt: new Date(new Date().setDate(new Date().getDate() - 2)),
        },
        {
          tenantId: tenant.id,
          sourceType: "PR TIMES",
          mediaType: "OWNED", // 自社発信
          channel: "公式",
          title: "シリーズ累計2000万個突破！春の限定パッケージ発売",
          summary: "ロフト・プラザで先行発売開始。SNSキャンペーンも同時開催。",
          engagementCount: 120,
          publishedAt: new Date(new Date().setDate(new Date().getDate() - 5)),
        },
        {
          tenantId: tenant.id,
          sourceType: "Web Media",
          mediaType: "PAID", // 記事広告
          channel: "WWD JAPAN",
          title: "なぜCLAYGEは若年層にウケるのか？ヒットの裏側",
          summary: "成分へのこだわりと、SNSを起点としたUGC創出の戦略を解剖。",
          engagementCount: 850,
          publishedAt: new Date(new Date().setDate(new Date().getDate() - 10)),
        }
      ]
    }
  }
});
```

### デモ時のトークスクリプト

> 「例えばCLAYGEの場合、自社発信（OWNED）のプレスリリース直後に、インフルエンサー（EARNED）のエンゲージメントが跳ねているのがこのグラフから一目で分かります。御社の競合でも同じ分析がすぐに可能です」
